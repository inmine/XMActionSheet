# XMActionSheet
模仿微信底部弹框，一句话弹框，适配了iPhone X

![Platform](https://wx1.sinaimg.cn/mw690/e067b31fgy1flpoz1xr73j206e0duwem.jpg)

# 使用方法
```
[XMActionSheet xm_actionSheetWithActionTitles:@[@"拍摄",@"从手机相册选择"] actionHander:^(NSUInteger index) {
        NSLog(@"点击了 %ld",index);
    }];
```
